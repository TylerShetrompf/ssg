# A simple static site generator, written in Python #
This project takes markdown files placed in the content folder, converts them to html, and then launches a webserver serving those pages.

## How To Run: ##
**_Requires python3_**
1. Download the project to a location of your choosing
2. Open your terminal to the root of the project
3. `chmod +x main.sh`
4. Place desired markdown files in /content
5. `./main.sh`
